# css-playground
Customize styles and preview in real time with content-editable style tag.

```css
<style contenteditable="true"> </style>
```

Don't forget to set:
```css
style{
    display: block;
}
```

[style.css](/style.css) contains only the following minimal styling.
```css:style.css
*{
    margin: 0;
    padding: 0;
}

body{
    background-color: #333;
}


style{
    display: block;
}
```
